## Installation
1. Clone repo
2. Bundle Install

## Running Server locally
1. Rails s

## Specs
1. bundle exec rake

## Logging in
Login using the following credentials to view subscription videos:
 - username=test@test.com
 - password=password

## ENVs
The following environment variables are required to run this application:
 - ZYPE_API_KEY
 - ZYPE_CLIENT_ID
 - ZYPE_CLIENT_SECRET
